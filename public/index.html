从这里开始

Run Gallery in browsers supports VRML to show gallery.

Run Chain Viewer in modern browser for chained blockchain viewer.

说明

这个程序分为两个部分，分别是三维画廊和对应的区块链浏览器。三维画廊使用IE浏览器+Cortona 4.2插件 显示在支持3D的显示设备上（需要WinXP和旧版NVIDIA驱动支持，另文说明）。区块链使用普通的现代浏览器（在WinXP下建议360极速浏览器）展示

打开一个IE窗口，访问“Gallery”页面，如果Contona插件配置正常，则可以看到3D画廊

打开一个Chrome（或360）窗口，访问“Chain Viewer”页面，点击“Open Viewer Window”按钮，这时弹出一个新的标签页 展示区块链浏览器，请不要关闭第一个标签页，即“Chain Viewer Initializer”，它将控制区块链浏览器和画廊联动。

观众点击画廊中的图案，区块链浏览器将自动（延迟不超过2秒）跳转到对应的token信息页。
